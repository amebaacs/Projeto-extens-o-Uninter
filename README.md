## Objetivo
Este projeto tem como objetivo incentivar adolescentes a utilizarem a tecnologia de forma consciente para organizar a rotina de estudos. Para isso, foi desenvolvida uma cartilha digital e um planner de estudos utilizando ferramentas digitais.

- Planner de Estudos (Google Sheets)
- Fotos da aplicação do projeto
